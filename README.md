![example](https://i.imgur.com/5eIA5ru.png)


## 參考文獻

連結的取得及影片解碼均參考 [c0re100/BahamutAnimeDownloader](https://github.com/c0re100/BahamutAnimeDownloader/)

需要下載 [FFmpeg解碼器](https://ffmpeg.zeranoe.com/builds/) 放置到主程式同一資料夾裡

將影片的連結輸入到搜尋框 便可進行下載
